-   [x] [Commitizen](https://github.com/commitizen/cz-cli) The commitizen command line utility.

## Start

```bash
# 0. Clone project
git clone https://github.com/yugasun/web-component-starter

# 1. Install dependencies
pnpm install

# 2. Start develop server
pnpm dev

# 3. Build
pnpm build
```
